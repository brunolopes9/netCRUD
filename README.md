# CRUD App with .NET, SQL Server, and React

This is a simple CRUD (Create, Read, Update, Delete) application built with:

- **Back-end:** .NET Web API  
- **Database:** SQL Server  
- **Front-end:** React with Axios and Bootstrap

## Features

- Add new employees
- View employee list
- Edit employee details
- Delete employees
- Toggle employee active status

## Technologies Used

- .NET Web API
- Entity Framework Core
- SQL Server
- React
- Axios
- React-Bootstrap
- React-Toastify
